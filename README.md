
Práctica P2 - Aplicación deliberadamente vulnerable
---------------------------------------------------
Contenido:
- app.py          : aplicación Flask vulnerable
- templates/      : páginas HTML
- uploads/        : carpeta para archivos subidos
- requirements.txt: dependencias (Flask)
- REPORT.md       : informe con vulnerabilidades, explotación y mitigaciones

Ejecución:
1) Tener Python 3.8+ instalado.
2) Crear un entorno virtual (recomendado) y activar.
3) pip install -r requirements.txt
4) python app.py
5) Acceder a http://localhost:5000

Nota: la aplicación es intencionalmente insegura para fines didácticos. No desplegar en producción.
